# Redis基础依赖库
存在一种场景，需要开发多个不同处理不同业务的依赖库，使用同一套redis配置。
业务若需要在redis基础依赖库的基础上，开发自己的业务信息，需要另外开发Redis业务依赖库。详见下文。
