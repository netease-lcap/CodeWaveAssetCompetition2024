# IPUtil

获取本机IP的工具类

## 逻辑详情

### getHostName

获取当前主机的名称  
入参：无  
出差：String hostName  

### getInetIPv4Address

获取当前系统中所有的IPv4地址列表  
入参：无  
出差：List<String> inetIPv4Address  

### getExternalIP

获取当前主机的外网IP地址  
入参：无  
出差：String externalIP
