# JsonUtil依赖库

提供JSON字符串解析，XML字符串转为JSON字符串逻辑，并支持JSONPath提取值

## 逻辑详情

### xmlToJson

将XML格式的字符串转换为JSON格式的字符串  
入参：String xmlStr  
出参：String  

### queryJson

从JSON字符串中查询指定键值对应的值  
入参1：String json JSON字符串  
入参2：String key 需要查询的JsonPath
